# Chladni Visualizer — クラドニ図形オーディオビジュアライザー

スマホ・PCのブラウザで動く、音に反応するクラドニ図形（砂が定在波の節線に集まる現象）のシミュレーターです。**ビルド不要**で、ブラウザで `index.html` を開くだけで動きます。

## 機能

- 🎤 **マイク入力** — 拾った音の周波数に合わせて図形がリアルタイムに変化（スマホ・PC両対応）
- 🖥️ **タブ音声キャプチャ** — **YouTube等を流しながらリアルタイム反映**（PC版 Chrome / Edge）
- ✋ **手動操作** — 周波数・粒子数・振動の強さを**スライダー＋数値入力**で調整
- 🎹 **サンプル音源** — ピアノ / バイオリン / フルート / チェロ（Tone.js）
- ⏺ **録画** — 映像を `.webm` で保存、現在の画を `.png` で保存
- 📱💻 **レスポンシブ** — スマホは縦並び、PCは横並びレイアウト

## 使い方

### ローカルで開く

ES モジュールを使うため、`file://` ではなくローカルサーバー経由で開きます。

```bash
# どれか一つ
python3 -m http.server 8000
# または
npx serve .
```

ブラウザで `http://localhost:8000` を開きます。

### YouTube をリアルタイム反映する（PC）

1. PC版 Chrome / Edge で開く
2. 「🖥️ タブ音声」ボタンを押す
3. 共有ダイアログで **YouTube のタブ**を選び、**「タブの音声を共有」にチェック**
4. YouTube を再生すると、音に合わせて図形が変化します

> スマホや非対応ブラウザでは「🎤 マイク」モードで、YouTube をスピーカーで流して拾う方法が使えます。

## 仕組み

正方形プレートの定在波変位場（座標 x,y ∈ [0,1]）：

```
z(x,y) = sin(n·π·x)·sin(m·π·y) + sin(m·π·x)·sin(n·π·y)
```

砂粒は変位の大きい（よく揺れる）場所から弾かれ、`|z| ≈ 0` の**節線**に集まります。検出した周波数からモード数 `m, n` を決め、図形を変化させています。

## ファイル構成

```
index.html          画面とUI
css/style.css       スタイル（レスポンシブ）
js/
  main.js           全体の制御・描画ループ
  chladni.js        変位場の計算とパーティクル物理
  audio.js          マイク／タブ音声入力と周波数検出
  sampler.js        サンプル音源（Tone.js）
  renderer.js       Canvas 描画
  recorder.js       録画・画像保存
```

## 動作環境

- マイク・録画：モダンブラウザ全般（スマホ含む）
- タブ音声キャプチャ：PC版 Chrome / Edge（`getDisplayMedia` の音声共有対応ブラウザ）
- HTTPS または localhost で動作（マイク等のAPIの制約）
